# POS管理系统 - PRD文档

## 项目概述
参考秘奥POS系统开发的Web版POS管理系统，支持多店铺、多仓库、网店管理。

## 用户画像
1. **系统管理员** - 管理整个系统，配置门店、仓库、用户权限
2. **门店经理** - 管理门店日常运营，查看销售报表
3. **收银员** - 前台收银，处理日常销售
4. **仓库管理员** - 管理库存，处理采购入库
5. **网店客户** - 在线浏览商品，下单购买
